# SchulteTables

Program for working with Schulte tables.<br>
<br>
How does it work:
1) User input
2) table is generated
3) the timer starts running
4) after completion, the execution time is displayed
5) the data is written to the table

Used:
OOP
functions
files
random
time
generators
